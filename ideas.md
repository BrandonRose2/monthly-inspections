# Monthly Inspections — Design Brief

## Chosen Approach: Clean Corporate Operations Dashboard

### Design Movement
Corporate utility meets modern clarity — inspired by enterprise operations tools (Notion, Linear, Airtable) with a no-nonsense, high-legibility aesthetic.

### Core Principles
1. **Legibility first** — every property name and status box must be instantly scannable
2. **Dense but breathable** — fit all 41 properties on one page without feeling cramped
3. **Action-oriented** — the two status boxes (✓ and ✗) are the heroes of each row
4. **Professional authority** — dark navy header, crisp white content area, strong typographic hierarchy

### Color Philosophy
- **Navy header** (#1a2744 / deep slate-navy) — authority and brand identity
- **White content** — clean, printable, professional
- **Green accent** for checkmark state — universally understood as "pass/complete"
- **Red accent** for X state — universally understood as "fail/issue"
- **Subtle row stripes** — alternating very light gray to aid scanning across 41 rows

### Layout Paradigm
Two-column region layout (matching the PDF reference), all on one scrollable page. Region headers act as visual anchors. Properties flow in a clean list under each region header.

### Signature Elements
1. Dark navy region header bars with white uppercase text (matching PDF)
2. Two distinct clickable boxes per property: ✓ (green on click) and ✗ (red on click)
3. Progress counter at the top showing "X of 41 inspected"

### Typography System
- **Header title**: Georgia or Playfair Display — editorial authority
- **Subtitle/meta**: System sans-serif, muted
- **Region labels**: Bold uppercase, white on navy
- **Property names**: Medium weight, dark charcoal

### Brand Essence
Internal operations tool for property management — reliable, clear, efficient.

### Signature Brand Color
Deep navy: #1e2d4a

## Style Decisions
- Checkmark box turns green with ✓ on click; X box turns red with ✗ on click
- Clicking again toggles off (deselects)
- Both boxes can be active simultaneously (e.g., inspected but with issues)
- Progress bar shows completion across all 41 properties
- Print-friendly: clean white background, boxes render clearly
